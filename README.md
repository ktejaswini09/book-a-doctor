# Book-A-Doctor

A MERN doctor appointment booking application built using React, TypeScript, Node.js, Express.js, and MongoDB.

## Features
- User Registration & Login
- Doctor Dashboard
- Patient Dashboard
- Appointment Booking
- Admin Dashboard

## Technologies Used
- React
- TypeScript
- Node.js
- Express.js
- MongoDB

## How to Run

### Frontend
```bash
npm install
npm run dev
```

### Backend
```bash
cd server
npm install
npm run dev
```
